# Reporting Security Vulnerabilities

If you discover a security vulnerability in this project, we encourage you to report it to us responsibly. Please follow the steps below:

1. **Do Not Create a Public Issue**: Please do not disclose the vulnerability publicly until it has been resolved to prevent potential exploitation.

2. **Provide Detailed Information**: Send an email to [security@charlux.com](mailto:security@charlux.com) with the following information:
   - Detailed description of the vulnerability
   - Steps to reproduce the vulnerability
   - Any relevant screenshots or logs that can help us understand the issue

3. **Communication**: We commit to responding to your report within [insert your response timeline, e.g., 72 hours]. If you report a vulnerability, you can expect to hear back from us regarding the progress made in resolving it.

4. **Join Us in Securing the Project**: After we have addressed the issue, we appreciate your help in communicating the fix to ensure that the wider community benefits from improved security practices.

Thank you for helping keep this project secure!
